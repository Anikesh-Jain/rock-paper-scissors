# Rock Paper Scissors

A fun browser-based Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features

- Interactive gameplay against computer
- Random computer move generation
- Score tracking
- Instant result display
- Responsive user interface
- Smooth browser gameplay

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Purpose

This project was created to practice JavaScript fundamentals, DOM manipulation, event handling, and interactive frontend development through a classic game project.

## Learning Outcomes

Through this project, I practiced:

- DOM manipulation
- Event handling
- Conditional logic
- Random number generation
- Score management
- Interactive UI development

## Future Improvements

- Sound effects
- Better animations
- Persistent score using local storage
- Dark mode
- Improved UI design
